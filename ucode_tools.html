Microcode Tools

All of these tools run on VAX/VMS (only). The .exe files are VAX executables, not Windows executables! .pli files are PL/1, and .pas files are Pascal.

  • calloc1.pat -- CVAX allocator pass 1 patterns
  • calloc1.pli -- CVAX allocator pass 1
  • calloc2.pas -- CVAX allocator pass 2
  • calloc3.pat -- CVAX allocator pass 3 patterns
  • calloc3.pli -- CVAX allocator pass 3
  • calloc4.pas -- CVAX allocator pass 4
  • iplatext.pli -- MicroVAX IPLA to text utility
  • micro2.exe -- microassembler executable
  • ralloc1.pat -- Rigel allocator pass 1 patterns
  • ralloc1.pli -- Rigel allocator pass 1
  • ralloc2.pas -- Rigel allocator pass 2
  • ralloc3.pat -- Rigel allocator pass 3 patterns
  • ralloc3.pli -- Rigel allocator pass 3
  • rvalloc1.pat -- Raven allocator pass 1 patterns
  • rvalloc3.pat -- Raven allocator pass 3 patterns
  • rvalloc3.pli -- Raven allocator pass 3
  • ualloc1.pat -- MicroVAX allocator pass 1 patterns
  • ualloc1.pli -- MicroVAX allocator pass 1
  • ualloc2.pli -- MicroVAX allocator pass 2
  • ualloc3.pat -- MicroVAX allocator pass 3 patterns
  • ualloc3.pli -- MicroVAX allocator pass 3
  • ualloc4.pli -- MicroVAX allocator pass 4
  • uldtotext.pli -- uld to text utility
